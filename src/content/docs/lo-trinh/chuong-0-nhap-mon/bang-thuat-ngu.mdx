:::tip[👉 Bài này bạn sẽ hiểu được]
- Nghĩa tiếng Việt ngắn gọn của những thuật ngữ tiếng Anh hay gặp nhất trong POE2.
- Cách "đọc hiểu" màn hình game khi gặp một từ tiếng Anh lạ.
- Nhóm thuật ngữ nào liên quan tới phần nào của game (chiến đấu, đồ đạc, endgame…).
:::

POE2 dùng **rất nhiều thuật ngữ tiếng Anh**, kể cả trong cộng đồng người Việt. Bạn **không cần dịch chúng ra khi nói chuyện** — quen rồi thì cứ dùng tiếng Anh cho gọn. Mục tiêu của bảng này là: khi bạn thấy một từ trong game hoặc trong guide, bạn **biết ngay nó nghĩa là gì**.

Đây là bảng **nền tảng** — gom các từ quan trọng nhất từ mọi chương lại một chỗ để bạn tra nhanh. Mỗi chương sau sẽ có bảng thuật ngữ riêng đi sâu hơn. Bạn **không cần học thuộc** — cứ chơi rồi quay lại tra khi cần. Mình chia theo nhóm cho dễ nhớ.

## Nhóm 1 — Tổng quan & nhân vật

<Hop variant="thuatngu">

| Tiếng Anh | Tiếng Việt / Giải thích |
|-----------|------------------------|
| ARPG | Game nhập vai hành động (giết quái, nhặt đồ, mạnh dần lên) |
| Build | Cách xây dựng nhân vật: kỹ năng + cây kỹ năng + trang bị |
| Meta | Những build/cách chơi đang được coi là mạnh nhất hiện tại |
| Class | Lớp nhân vật, chọn lúc tạo nhân vật |
| Ascendancy | Lớp nâng cao của class — mở khóa trong game, định hình build |
| Level (lvl) | Cấp độ nhân vật |
| Passive tree | Cây kỹ năng bị động — cộng điểm để tăng chỉ số |
| Respec | Hoàn điểm để cộng lại cây kỹ năng theo cách khác |
| Stats | Các chỉ số / thuộc tính của nhân vật và đồ |

</Hop>

## Nhóm 2 — Chiến đấu & kỹ năng

<Hop variant="thuatngu">

| Tiếng Anh | Tiếng Việt / Giải thích |
|-----------|------------------------|
| Skill | Kỹ năng / chiêu thức |
| Skill gem | Viên ngọc tạo ra một kỹ năng chủ động để dùng |
| Support gem | Viên ngọc hỗ trợ — gắn vào skill gem để tăng/đổi hiệu ứng |
| Uncut Skill Gem | Ngọc kỹ năng "thô", khắc ra để chọn kỹ năng bạn muốn |
| DPS | Sát thương mỗi giây (đo độ mạnh đầu ra của build) |
| Tank / Tanky | Nhân vật "trâu", chịu đòn tốt, khó chết |
| Clear speed | Tốc độ dọn sạch quái trong một khu vực |
| Buff | Hiệu ứng làm mạnh lên (cho mình) |
| Nerf | Bị chỉnh yếu đi (thường do patch) |
| Debuff | Hiệu ứng làm yếu đi (thường đặt lên kẻ địch) |
| Single target | Sát thương dồn vào một mục tiêu (dùng đánh boss) |
| AoE (Area of Effect) | Sát thương diện rộng (dọn nhiều quái cùng lúc) |

</Hop>

## Nhóm 3 — Đồ đạc, loot & chế đồ

<Hop variant="thuatngu">

| Tiếng Anh | Tiếng Việt / Giải thích |
|-----------|------------------------|
| Loot | Đồ rơi ra khi giết quái |
| Item | Vật phẩm / món đồ |
| Gear | Trang bị nhân vật đang mặc |
| Rarity | Độ hiếm của đồ (Normal → Magic → Rare → Unique) |
| Normal | Đồ thường (trắng), không có thuộc tính phụ |
| Magic | Đồ xanh dương, có 1–2 thuộc tính phụ |
| Rare | Đồ vàng, nhiều thuộc tính phụ — xương sống của build |
| Unique | Đồ huyền thoại (cam), có hiệu ứng đặc biệt riêng |
| Affix | Thuộc tính phụ ngẫu nhiên trên đồ (prefix/suffix) |
| Currency | Vật phẩm tiền tệ — vừa là tiền, vừa dùng để chế đồ |
| Orb | Một loại currency phổ biến (vd: Exalted Orb, Chaos Orb…) |
| Craft | Chế / chỉnh sửa thuộc tính của đồ |
| Vendor | NPC bán đồ trong thành |
| Trade | Mua bán đồ với người chơi khác |
| Stash | Rương cất đồ chung của tài khoản |

</Hop>

## Nhóm 4 — Endgame & vận hành mùa giải

<Hop variant="thuatngu">

| Tiếng Anh | Tiếng Việt / Giải thích |
|-----------|------------------------|
| Campaign | Phần cốt truyện chính |
| Endgame | Nội dung sau khi xong cốt truyện (chơi lâu dài) |
| Map | Bản đồ endgame để chạy đi chạy lại |
| Waystone | Vật phẩm dùng để mở/kích hoạt một map endgame |
| Atlas | Bản đồ thế giới endgame, gắn với hệ thống Maps |
| Boss | Quái trùm, mạnh hơn quái thường |
| League | Mùa giải — sân chơi mới, có thời hạn |
| Standard | Sân chơi vĩnh viễn, không reset |
| League mechanic | Cơ chế đặc trưng riêng của một league |
| Economy | Nền kinh tế giữa người chơi |
| Farm | Cày một nội dung lặp lại để kiếm đồ/currency |
| Patch | Bản cập nhật của game |
| HC / SC | Hardcore (chết là mất nhân vật) / Softcore (chết hồi sinh lại) |
| HP / ES / Mana | Máu / Khiên năng lượng / Năng lượng để dùng kỹ năng |

</Hop>
